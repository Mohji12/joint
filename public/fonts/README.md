# Fonts folder

## Westack Regular (JOINTLY hero text)

Place the **Westack Regular** font file here with one of these names:

- `Westack-Regular.ttf`
- `Westack-Regular.woff2`
- `Westack-Regular.woff`

If your file has a different name (e.g. `Westack.ttf`), rename it to `Westack-Regular.ttf`.

**Download:** Westack is available from [Fonts2u](https://fonts2u.com/westack.font), [FontMirror](https://www.fontmirror.com/westack), or [Envato Elements](https://elements.envato.com/westack-modern-serif-V88L3EL). Check licensing for commercial use.
